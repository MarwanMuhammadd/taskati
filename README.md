# Taskati 📋

Taskati is a modern task management app built with Flutter to help users organize their daily tasks, track progress, and improve productivity through a clean and intuitive user interface.

The app focuses on simplicity, performance, and a smooth user experience.

---

## ✨ Features

- Add and manage daily tasks easily
- Track in-progress tasks
- Mark tasks as completed
- Clean and modern UI design
- Smooth and responsive performance
- Well-structured and scalable codebase
